A simple FIGlet web service

I got tired of trial and error with FIGlet, so I made this simple service. It's built from the excellent Javascript FIGlet parser by Scott Gonzalez.

View the library

You can demo all available fonts at /browse. Enjoy!

Options

You can pass the following query string parameters:

  • Text: Content to be displayed
  • Font: Font to be displayed

Example
?text=foobar&font=isometric1
